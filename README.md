## 使用方法
Fork后直接点击Run即可。由于窗口大小限制，Replit的Webview窗口无法正常显示其网页，请将Webview窗口地址栏内的网址复制，在浏览器内打开。初始管理员密码是**12345678**。

修改密码只需去Deploy文件夹下找到`Config.xml`文件，修改其中的`LoginPassword`项即可。

Fork & Run.
